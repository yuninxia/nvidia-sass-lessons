# Useful Links

## CUDA Binary Tools
- **CUDA Binary Utilities** (nvdisasm, cuobjdump): https://docs.nvidia.com/cuda/cuda-binary-utilities/
- **CUDA Binary Utilities PDF** (archived for quick CLI reference): https://docs.nvidia.com/cuda/archive/9.1/pdf/CUDA_Binary_Utilities.pdf
- **PTX ISA Documentation**: https://docs.nvidia.com/cuda/parallel-thread-execution/

## GPU Profiling & Analysis Tools
- **Nsight Compute** (Kernel profiler with SASS analysis): https://developer.nvidia.com/nsight-compute
- **Nsight Systems** (System-wide profiler): https://developer.nvidia.com/nsight-systems
- **CUPTI** (Profiling Tools Interface): https://developer.nvidia.com/cupti
- **Nsight Compute Documentation**: https://docs.nvidia.com/nsight-compute/ProfilingGuide/

## Binary Instrumentation
- **NVBit** (Dynamic Binary Instrumentation Framework): https://github.com/NVlabs/NVBit
- **NVBit Research Paper**: https://research.nvidia.com/publication/2019-10_nvbit-dynamic-binary-instrumentation-framework-nvidia-gpus
- **NVBitFI** (Fault Injection Tool): https://github.com/NVlabs/nvbitfi
- **NVBit Tools Examples**: https://github.com/CoffeeBeforeArch/nvbit_tools

## Unofficial SASS Assemblers
- **MaxAs** (Maxwell architecture): https://github.com/NervanaSystems/maxas
- **CuAssembler** (All generations): https://github.com/cloudcores/CuAssembler
- **TuringAs** (Volta/Turing): https://github.com/daadaada/turingas
- **cuasm** (Alternative assembler): https://github.com/gpuocelot/cuasm

## GPU Architecture Documentation
- **CUDA Programming Guide**: https://docs.nvidia.com/cuda/cuda-c-programming-guide/
- **GPU Architecture Whitepapers**: https://www.nvidia.com/en-us/data-center/resources/
- **CUDA Toolkit Documentation**: https://docs.nvidia.com/cuda/
